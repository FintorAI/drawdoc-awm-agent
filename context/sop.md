**Process Overview **
 Systems: 

1. Encompass  ‐  Login to Encompass App 
2. Outlook 365  ‐  Outlook 365 Login

 The Docs drawing process can be best summarized into below steps, 

1. Verification of Documents 
2. Validation and Updation of Accurate data in Encompass 
3. Validate Accurate data in Encompass 
4. Prepare Closing Package 

 Process Flow: 

1. Once a file is CTC (Clear to Close) by UW then file hits to Docs Ordered queue in 
    Encompass and CD status in Encompass must says CD Approved in order to start DOCS 
    Draw. 
2. Docs Team lead to monitor Docs Ordered Queue is approved by UW or not (UW CTC/ 
    UTR), if the CD Status is approved and acknowledged by Borrower/ALL or not before 
    assigning the loan to team members. 
3. Team lead needs to review “Docs Ordered” queue and assign loans to respective team 
    members. Team members then work on these loans on the basis of CD acknowledge 
    time/closing time/date/priority. 
4. Provide data inputs / validation of required information in Encompass while verifying 
    this data with available docs in Encompass (Documents + Encompass) 
5. Validation  of  required  information  in  Encompass  while  verifying  this  data  with 
    available docs in Encompass. 
6. Input  “at  closing”  conditions.  Verify  all  fees  and  Order  AUDIT /  Compliance 
    (Encompass) 


7. Pass‐ If Yes, Send Closing Package to Title Company via DIRECT.
8. Pass‐ If No, Notify Supervisor with all audit error details. Once it is cleared/short out 
    then Send Package to Title Company via Encompass. 

 **Process TAT: 6**  Business Hours from the time loan hits to DOCS order queue and all the Closing 
Disclosure Status is updated as CD approved & ACK by Borrower(s). 

 **NOTE:**  All Concerned Parties & Management to be informed 2 hours in advance regarding any 
delay on files.

1. On Texas properties the Deed of Trust need not be drafted with additional verbiage like “Joint

       (i) The borrowers’ marital status will determine whether a spouse’s signature is required on 
          the Deed of Trust to create a valid lien on homestead under Texas law

```
order to create a joint tenancy with right of survivorship 
```
```
(iii) Creating a joint tenancy with right of survivorship is contrary to the common practice in 
Texas,  and  may  impact  the  borrowers’  estate  planning  objectives  without  their 
knowledge
```

```
(iv) The manner in which title is held by the borrowers has no impact on the validity of the
```
